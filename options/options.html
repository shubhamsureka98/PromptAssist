PromptAssist

Optimize your prompts before sending. Manual trigger only — never auto-sends.

Mode

API (default): an LLM rewrites your draft via your key. Clean, no chat clutter. In this chat: asks the chat you're in to rewrite, then offers Replace, Insert, or Open in a new chat.

API provider

Different models trade speed vs quality. Claude Haiku 4.5 is the default — fast, cheap, good at structured rewrites.

Style

Auto: chooses based on your draft. Short/simple → refine. Long/complex → structure (adds role, context, task, format). Structure always applies best-practice prompt engineering from Claude, OpenAI & Gemini.

Disable on sites

Token usage

Tracks how many tokens PromptAssist uses from your API key each month — not the LLM platform's own counter. The bar shows in the floating button menu as you work.
